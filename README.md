A comprehensive support helper tool with World Clock features, meeting scheduling capabilities, and custom integrations, Clipboard History, Standard Snippets.

## Installation

1. Clone this repository
   or
   Download the Zip:
   <img width="403" height="307" alt="image" src="https://github.com/user-attachments/assets/67ea6dbb-d05b-4ba4-bb2d-f46ee198a603" />

3. Install required dependencies:
   Install Python and set the appropriate Env Variable path for python
   and then use the following command to install the required modules
   ```bash
   pip install -r requirements.txt
   ```
5. Run the main application:
   ```bash
   python main.py
   ```
   Or make a shortcut and put in the startup for Auto Start on logon
